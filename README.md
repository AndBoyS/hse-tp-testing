## Запуск

Для использования рекомендуется сначала создать и активировать виртуальное окружение:

Windows:
```bash
python -m venv .venv
.venv\Scripts\activate.bat
```

Linux/Mac:
```bash
python -m venv .venv
.venv/bin/activate
```

Далее нужно установить необходимые библиотеки:
```bash
pip install -r requirements.txt
```
